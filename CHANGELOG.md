# CHANGELOG.md

## [v2.4.1] - 2023-10-15
### Added
- **Game-Specific Profiles**: Introduced customized profiles for popular games like “World of Warcraft”, “Fortnite”, and “League of Legends”, allowing tailored click settings for each title.
- **Hotkey Configurations**: Added advanced hotkey configurations for seamless switching between click modes during gameplay.

### Changed
- **Performance Enhancements**: Optimized the click timing engine for smoother performance, reducing lag during high-action sequences.
- **User Interface**: Updated the UI for improved navigation and accessibility, especially for gamers with multi-monitor setups.

### Fixed
- Resolved an issue where the auto clicker would sometimes freeze when launched alongside certain anti-cheat software.
- Fixed a bug that caused click rate settings to revert to defaults after a game session ended.

## [v2.3.0] - 2023-07-22
### Added
- **Multi-Click Mode**: Introduced a new multi-click mode that allows for simultaneous clicks across multiple areas of the screen, perfect for RTS and simulation games.
- **Click Patterns**: Implemented a feature to create and save custom click patterns for repetitive tasks in games such as “Starcraft II” and “The Sims”.

### Changed
- **Session History**: Enhanced session history tracking to log clicks accurately, providing players insights into their clicking habits.
- **Compatibility Updates**: Improved compatibility with Windows 11 and adjusted settings for better integration with new gaming features.

### Fixed
- Addressed an issue with DPI scaling settings that caused clicks to misalign in high-resolution displays.
- Fixed a minor bug that intermittently reset user configurations on restart.

## [v2.1.0] - 2023-03-10
### Added
- **Randomized Click Timing**: Implemented randomized click timing to help simulate human clicking behavior, particularly for online games that monitor repetitive actions.
- **In-Game Overlay**: Added an optional in-game overlay showing current click rates, providing real-time feedback during sessions.

### Changed
- **UI Refresh**: Made UI adjustments for a more streamlined experience and improved clarity on settings options.
- **Enhanced Documentation**: Updated manual and help sections to better guide users in setting up and customizing their click settings.

### Fixed
- Resolved a compatibility issue with specific gaming mice that caused false clicks
